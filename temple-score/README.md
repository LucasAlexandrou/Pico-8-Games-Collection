## Temple score
Dodge chasing and random moving enemies, use powerups, collect coins to increase your score.

## Gameplay
![Gameplay](screenshots/templescore_9.gif)

## Controls
- Typical Pico-8 Controls
- Arrow keys to move
- Z/X to traverse menus

## Features
- Highscore system
- Music + sounds
- Menu + settings + death screen
- Powerups that freeze enemies and give you increased speed
